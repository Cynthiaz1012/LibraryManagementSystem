##Project Description
This project is a Java-based Library Management System developed using Object-Oriented Programming (OOP) concepts.

The system allows users to:
- Borrow library items
- Return library items
- Search for items
- Generate reports
- Save data into CSV files

The project demonstrates:
- Inheritance
- Polymorphism
- Abstract classes
- Interfaces
- Exception handling
- File I/O
- Recursion
- Java Collections
- Stream

##Main Classes:
- Item (Abstract parent class for all library items)
  Subclasses:
    Book
    DVD
    Magazine
- User (Abstract parent class for all users)
  Subclasses:
    Student
    Teacher
    Admin
-Library
-Reportable (interface)
